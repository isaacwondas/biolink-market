# Biolink Market

A bio-link platform built with Next.js and Supabase, where users get a shareable profile page (`/yourusername`) with a QR code, and admins manage payments and accounts through a dedicated dashboard.

## Features

- Public profile pages at `/[username]`
- QR code generation for easy sharing
- Admin dashboard with protected routes
- User onboarding flow
- Partial payment tracking and admin ledger

## Tech Stack

- [Next.js](https://nextjs.org) 16
- [Supabase](https://supabase.com) (auth & database)
- [Tailwind CSS](https://tailwindcss.com) 4
- TypeScript
